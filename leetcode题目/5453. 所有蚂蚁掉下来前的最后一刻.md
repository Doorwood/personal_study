# 思路
可以通过画图轻松知道,触碰以后,可以当作互相穿过,所以还是找最大的,先左右排序,然后找到左边数组掉下需要的最大时间,
然后再找到右边数组掉下去最大的值,最后返回较大值.

class Solution {
    public int getLastMoment(int n, int[] left, int[] right) {
        int lMax = 0;
        int rMax =0;

        if(left.length!=0){
            Arrays.sort(left);
            lMax = left[left.length-1];
        }
        if(right.length!=0){
            Arrays.sort(right);
            rMax=n-right[0];
        }
        return Math.max(lMax,rMax);
    }
}
