# 思路:
动态规划
根据前面的状态推导出状态转移方程:f(n)=f(n)f(0)+f(n-1)f(1)+...+f(0)f(n)

# 实现代码
```java
class Solution {
    public int numTrees(int n) {
        return calculate(n);
    }
    public int calculate(int n){
        if(n==0){
            return 1;
        }
        if(n==1){
            return 1;
        }
        if(n==2){
            return 2;
        }
        int len =n/2+n%2;
        int ans=0;
        for(int i=1;i<=len;i++){
            if(n%2==1&&i==len){
                ans+=calculate(n-i)*calculate(i-1);
                continue;
            }
            ans+=2*(calculate(n-i)*calculate(i-1));
        }
        return ans;
    }
}
```
